Javascript Cyber Security Quiz-
A simple browser-based quiz application built with js that tests knowledge on fundamental cyber security terms.
Features-

Questions-Covers basic concepts like hacker types,firewalls,DDoS,ransomware,zero-day vulnerabilities.
Case-insensitive checking-User answers are normalized (trimmed and converted to lowercase) for flexible checking against the correct answer.
Score  tracking-Calculates and displays the final score.

This quiz runs directly in any modern web browser.
